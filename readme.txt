git commands
git status — что происходит
git add . — добавить все изменения в будущий коммит
git add images/. - добавить в будущий коммит все изменения в папке images
git add readme.txt - добавить в будущий коммит изменения файла readme
git commit -m '123' - сделать коммит (в Windows "123")
git commit --amend - докоммитить в предыдущий коммит
git diff - изменения в наблюдаемых файлах
